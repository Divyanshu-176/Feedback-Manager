# Feed Fizz

Community-driven feedback platform where users can post ideas, vote on requests, and follow the product roadmap.

Live demo: [https://feed-fizz.vercel.app/](https://feed-fizz.vercel.app/)

## Tech Stack

- Next.js
- React
- TypeScript
- Prisma
- Clerk Auth
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
